# Werf

Open-source AI coding agent.

- **Website:** [werfagent.com](https://werfagent.com)
- **Source:** [werfagent/werfagent](https://github.com/werfdesktop/werfdesktop)

## Overview

Werf is a coding agent that can plan, edit, and run code in your projects. It is free to use and developed in the open.

## Getting started

See [werfagent.com](https://werfagent.com) for installation and docs.
